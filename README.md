Eshu
====

A CLDR(Common Locale Data Repository) library.

Introduction
------------

Eshu is a CLDR(Common Locale Data Repository) library.

### Who is Eshu?

Eshu(Èṣù) is the god from the religion of the Yoruba people who can speak all languages.
